## 语法

开发时，如果想要找一些模板，可以按照下面的方式，用一些语法和工具，可以更加迅速的找到自己想要Copy的模板(KPI!!!)，狗头保命

### Fofa
```
org=“” && title=“**” && body=“***”
```

### Google语法
- \*\*\*图像
```
site:.***.***.*** filetype:*** | *** | *** -*
```

- 关键字
```
site:.***.***.*** intext:"***" filetype:***
```

- 边缘资产
```
***(主关键词) '***'（可以更换与想了解的知识相关的东西）(次要) '***'（更换为系统、管理、后台、登陆、注册等）(再次)
```
