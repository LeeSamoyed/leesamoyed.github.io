# C++ Primer
## 序论

目标

```
书店保存所有销售记录的档案，每条记录保存了某本书的一次销售的信息(一册或多册)。每条记录如下
0-201-70353-x   4   24.99
```

需求

```
1. 定义变量
2. 进行输入与输出
3. 使用数据结构保存数据
4. 检测两条记录是否有相同的ISBN
5. 包含一个循环来处理销售档案中的每条记录
```
